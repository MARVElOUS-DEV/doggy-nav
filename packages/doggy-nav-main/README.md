## 安装部署
此服务基于Nuxtjs, 可以使用SSR或者CSR部署方式；使用SSR部署方案时，项目启动方式为nodejs项目；使用CSR部署方案时，项目为常规静态web部署，即编译为js，css，html文件后部署到服务器中；
